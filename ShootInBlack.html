Shoot In Black

Présentation du projet


Nous avons créé un shoot'em up nommé Shoot In Black En référence au film Men In Black.
Le but du jeu est donc de tirer sur les extra-terrestres pour les empêcher d'envahir la Terre.

Technologies utilisées

  • JavaScript

  • HTML

  • CSS
