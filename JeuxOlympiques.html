Présentation du projet


Avec mon binôme, nous avons dû exploiter une base de données sur tous les Jeux Olympiques ayant eu lieu jusqu'à présent.
Nous avons créé des requêtes pour récupérer des données de la base, et pour cela, nous l'avons ventilée afin d'optimiser son traitement.

Technologies utilisées

  • PostgreSQL

Compétences et savoir-être développés


  • Travail d'équipe

  • Communication

  • Créativité

  • Gestion d'une base de données

  • Création et optimisation des requêtes

  • Analyse statistique des données
