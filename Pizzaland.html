Présentation du projet


Par groupe de 2, nous avons créé une API REST qui a pour but de gérer l'entreprise Pizzaland.
Cette API permet la gestion des commandes, des pizzas, de leurs types de pâtes ainsi que de leurs ingrédients.
Pour cela, nous avons utilisé, les méthodes GET, POST, PATCH et DELETE.

Technologies utilisées

  • Java

  • PostgreSQL
